# Debug

The Debug (`debug`) application provides several additional tools that can be useful for developers working on debugging issues in an IncusOS system:

* `htop`
* `ifstat`
* `iputils-ping`
* `jq`
* `mtr-tiny`
* `nano`
* `net-tools`
* `netcat-openbsd`
* `procps`
* `sqlite3`
* `strace`
* `tcpdump`
